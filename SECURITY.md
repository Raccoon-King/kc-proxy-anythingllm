# Security Policy

## Supported Versions
This project only supports the latest release on the `main` branch.

## Reporting a Vulnerability
If you discover a security issue, please do not open a public issue.

Send a report to:
- GitHub: open a private security advisory on this repository

Please include:
- A clear description of the issue and impact
- Steps to reproduce
- Any relevant logs or configuration details (redact secrets)

We will acknowledge receipt and work on a fix as quickly as possible.
